## Basic Python Concepts 

This repository contains two simple Python scripts demonstrating basic concepts like user input, mathematical operations, and string concatenation.

## Task 1: Basic Mathematical Operations
**File:** `index.py`  
This program prompts the user for two numbers and calculates their addition, subtraction, multiplication, and division.

## Task 2: Personalized Greeting
**File:** `greeting.py`  
This program asks for the user's first and last names, combines them, and prints a friendly greeting message.
